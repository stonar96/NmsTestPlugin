Commands and required permissions:
All commands require at least the permissions 'minecraft.command.nmstestplugin' and 'nmstestplugin.command.nmstestplugin'.

Command:
/nmstestplugin reload
Description:
Reloads the config.yml.
Permissions:
nmstestplugin.command.nmstestplugin.reload

Command:
/nmstestplugin text
Description:
Displays text.
Permissions:
nmstestplugin.command.nmstestplugin.text
